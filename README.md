## ⚠️ Notes

- **Ads** — Pluto TV is free because it's ad-supported. Expect commercials, just like regular TV.
- **US Content** — Channels are from the US Pluto TV catalog.
- **Sleep mode** — Hosted on Render free tier. First load after 15 minutes of inactivity may take ~30 seconds to wake up. Be patient — it will load!
- **Stream issues** — If a channel won't play, try another. Some channels have regional restrictions even within the US catalog.

```bash

cd https-Reddawn.xyz
npm install
npm start
```

Deploy free on [Render.com](https://render.com) — connect your GitHub repo and it deploys automatically.

## 🐛 Troubleshooting

**No streams / channel not loading**
- Wait 30 seconds and try again (server may be waking up)
- Try a different channel

**Wrong language / German content**
- This is fixed in the latest version. Re-install the addon.

**Addon not showing in Stremio**
- Make sure you're on Stremio v4.4+
- Try uninstalling and reinstalling
